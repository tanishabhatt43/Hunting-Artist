# Hunting-Artist

A web platform that connects independent artists with recruiters, allowing profile showcasing, enquiry management, and structured communication.

## Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript
- PHP
- MySQL
- XAMPP
- NetBeans IDE

## Features
- Dual login system for artists and recruiters
- Artists can create detailed profiles showcasing talents
- Recruiters can send enquiries with automated email notifications
- Artists can accept/reject enquiries and update status
